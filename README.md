# git-info-edn

A Leiningen plugin to put git info into a resource.

## Usage

Put `[git-info-edn "0.2.1"]` into the `:plugins` vector of your
`:user` profile.

Or, put `[git-info-edn "0.2.1"]` into the `:plugins` vector of your project.clj.

    $ lein git-info-edn
